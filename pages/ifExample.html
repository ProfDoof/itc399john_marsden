If Statement Example

You can construct an IF statement by placing the IF block and ENDIF block.

Any ELSE or ELSE IF block that goes with the IF statement you are creating goes between the IF and ENDIF blocks. It should have the same indentation as the IF and ENDIF blocks you wish to place it between. Don't forget to indent any blocks inside of the IF statement besides the ELSE and ELSE IF blocks over by 1.

You can change the boxes into the appropriate symbols for the operation you are attempting to perform by clicking on .
